##Neste código, criamos a classe Calculadora com métodos estáticos 
para cada operação matemática. No main(), pedimos ao usuário que insira 
os números e a operação desejada e, em seguida, usamos os métodos estáticos 
da classe Calculadora para realizar o cálculo e exibir o resultado. 
Certificando-se de tratar a divisão por zero.
